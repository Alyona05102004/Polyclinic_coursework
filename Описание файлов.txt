bd.php - файл с подключением к бд
edit_doctor_modal.php - модальное окно для редактирования данных о враче
education_operations.php - sql запросы для вывода информации об образовании

get_doctor.php - запрос для вывода информации о враче для редактирования
get_polyclinic.php - запрос для вывода списка поликлиник при выбранном отделении
index.php - главная страница
make_new_doctor.php - функция модального окна для создания новой записи докторов
qualification_operations.php - sql запросы для вывода информации о повышении квалификации
save_new_doctor.php -  sql запрос для сохранения данных о докторе
