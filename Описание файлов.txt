get_department.php - sql запрос для вывода списка отделений, если выбрана поликлиника
get_doctor.php - запрос для вывода информации о враче для редактирования
get_patient.php - получить информацию для формы с личной информацией пациента

save_new_doctor.php -  sql запрос для сохранения данных о докторе
